\# 🏠 RoomieSync

![CI Pipeline](https://github.com/gittam06/RoomieSync/actions/workflows/ci.yml/badge.svg)


\*\*RoomieSync\*\* is a full-stack household management application designed to bring peace to shared living spaces. It helps roommates manage finances, chores, and communication in one unified dashboard.



\## 🚀 Key Features



\### 💸 Money Management

\- \*\*Smart Splitting:\*\* Automatically splits bills and tracks who owes whom.

\- \*\*Debt Redistribution:\*\* When a user leaves, their debts are automatically recalculated and distributed among remaining members.

\- \*\*Settlements:\*\* One-click settlement recording.



\### ✅ Gamified Tasks ("Chore Wars")

\- \*\*Karma Points:\*\* Earn points by completing chores to climb the house leaderboard.

\- \*\*Groceries \& Inventory:\*\* Shared shopping list and stock tracking for household supplies.



\### 💬 Social Hub

\- \*\*House Chat:\*\* Real-time group chat for the household.

\- \*\*Polls:\*\* Vote on dinner plans, movies, or house rules.

\- \*\*Anonymous Complaints:\*\* A safe space to raise issues without conflict.



\## 🛠️ Tech Stack

\- \*\*Frontend:\*\* React (Vite), Tailwind CSS, Lucide Icons

\- \*\*Backend:\*\* Python FastAPI, SQLAlchemy

\- \*\*Database:\*\* PostgreSQL (Production) / SQLite (Development)

\- \*\*Deployment::\*\* Render (Backend), Vercel (Frontend)
